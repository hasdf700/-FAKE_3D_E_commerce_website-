# NEXUSMall - 3D 虛擬商城網站

這是一個結合前台視覺效果與後台資料庫管理的多功能電子商務網站專案。

## 🚀 技術棧 (Tech Stack)
* **前端 (Frontend)**: HTML5, CSS3, JavaScript (DOM 操作), Bootstrap
* **後端 (Backend)**: PHP
* **資料庫 (Database)**: MySQL

## 🌟 核心功能
* **動態店面展示**：使用 JavaScript 隨機且不重複地生成主題店面（咖啡廳、電玩店、花店等）。
* **資料庫串接**：利用 PHP 透過 SQL 指令，動態撈取並呈現城鎮與商店的關聯資料。
* **社群連結整合**：自動將店鋪卡片與外部社群平台（如 Facebook）進行動態綁定。

## 💻 如何在本機運行 (Installation & Setup)
1. 下載並安裝 [XAMPP](https://www.apachefriends.org/)。
2. 將此專案資料夾移至 `C:/xampp/htdocs/` 下。
3. 開啟 XAMPP Control Panel，啟動 **Apache** 與 **MySQL**。
4. 匯入資料庫檔案（ `expstore.sql`）。
5. 在瀏覽器輸入 `http://localhost/NEXUSMall/` 即可瀏覽網站。
